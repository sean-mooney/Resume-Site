CPMOONEY.COM

This website was a project I completed for my father who is a Construction Project Manager. Using SquareSpace, I created a platform for him to post new homes he's building, upload his portfolio to showcase his work, and have others contact him if they're interested. While SquareSpace provides an easier way to make websites, it has many limitations that require workarounds to perfect the product. One example is there is no feature for a portfolio to be on more than one page. To get around this, I hardcoded two buttons, a list of numbers, and wrote a script to determine what the next page should be based on the current page being viewed, and highlight the current page's number. It was a good demonstration of hacking around the limitations of a software and working around them to achieve a better result.
