CPMOONEY.COM

This website was a small project for my dad who is a Construction Project Manager. Using SquareSpace, I created a platform for him to post new homes he's building, upload his portfolio to showcase his work, and have others contact him if they're interested. While SquareSpace provides an easier way to make websites, it has many limitations that require workarounds to perfect the product. One example is there is no feature for a portfolio to be on more than one page. To get around this, I hardcoded two buttons, a list of numbers, and wrote a script to determine what the next page should be based on the current page being viewed, and highlight the current page's number. It was a good demonstration of learning the limitations of a software and working around them to achieve a better result.

Currency Converter

This was by far the most challenging web app I have ever created. Built for my Java 2 class at Norwalk Community College, I utilized advanced practices and techniques that made it almost impossible for stack overflow to help when I got stuck. The course had the highest drop rate of any in the school, we started the semester with around 20 students and dropped down to just 4 by the end. The app itself features a login page that queries an SQL server, has back door protection, converts a value based on a currency exchange rate retrieved from a server, retrieves the currencies available from the server and populates a drop down list with them, and much more. The whole project took all semester to complete, it was how we learned so much about Java, by building off an existing product. All of this was accomplished without anything being hardcoded, we would lose points if anything was. Link to the GitHub is below.
